Usage

To use this component, follow these steps:

1. Link the checked-items.css stylesheet to the document:

Add the following link element to the document head:

<link rel="stylesheet" href="/inc/checked-items/checked-items-1.css" />
2. Link the checked-items.js script to the document.

Add the following script element near the bottom of the page:

<script src="/inc/checked-items/checked-items-1.js" ></script>
3. Decorate the list elements with this attribute:
<dl class="checked-items">
<ul class="checked-items">
<ol class="checked-items">

The script generates input elements of type checkbox with unique storage keys, and adds change event handlers to persist the checkbox state in the browser's local storage.

Implementation Details

The script implements an anonymous immediately called function, which encapsulates the entire functionality without polluting the global namespace. Implemented as a non-module script, no symbols are exported, but the script can be imported like any other module script.
