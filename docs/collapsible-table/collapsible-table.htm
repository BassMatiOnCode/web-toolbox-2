Collapsible Table Rows and Columns

The table body height cannot be controlled with the style height, and fiddling around with the display style quickly leads to undesired side effects. Therefore the smooth height and width transition effects of the collapsible-structures component have been abandoned. The collapsible-table component is based on the visibility:collapse style, which works pretty well on table rows and columns.

Example

Click on a rowgroup header (decorated with a circled plus or minus) to toggle the visibility of the row group. All rows except the first (the group controller row) are affected.

Click on a column control header cell to toggle the visibility of a column. The entire column disappears, including the controller cell.

Click on the first column control header cell in that row. It has a different control icon, because the first column cannot be hidden. Instead, a click overrides the visibility of all columns without changing their state. Collapsed columns are now visible and show a circled plus icon, like rows. Their state is now accessible and can be toggled. When done, click the first column control cell again to collapse the columns with a circled plus.

Usage

To use the collapsible-table component, follow these steps:

  1. Download the files collapsible-table-1.js and collapsible-table-1.js from the corresponding web-toolbox-2 code subfolder on Github and store them in the include folder of your web project.

  2. Include the stylesheet in the HEAD section:

    <link rel="stylesheet" href="/inc/collapsible-table/collapsible-table-1.css" />

  3. Include the script at the bottom of the BODY:

    <script type="module" src="/inc/collapsible-table/collapsible-table-1.js"></script>

  4. Create a collapsible row group in a table body element and decorate the TBODY with the collapsible class name:

    	<tbody class="collapsible">
	    <tr class="row-header"><th colspan="8">Package</th></tr>
		<tr><td>QFN40</td><td>QFN48</td>⋯<td>PLCC68</td></tr>
		<tr><td>⋯</td></tr>
	</tbody>

    The first row of the table body element is considered the row group header and will be decorated by CSS rules with circled symbols that indicated the row group state.

  5. Create a COL element for every column in the table:

    	<table>
	<colgroup><col/><col/>⋯<col/></colgroup>
	⋯
	</table>

    These elements control the visibility of table columns.

  6. Decorate a controller row with the column-controller classname, This is typically the first heading row in the table:

    	<thead><tr class="column-controller">
		<th>...</th>...<th>...</th>
	</tr></thead>

    This classname causes a CSS rule to decorate the cells with circular column control icons.
